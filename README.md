## ❓ What it is

The Wordle Score Detector is an open source project. It operates on any operating system with python and can be ran as a simple python script or integrated into a server. The script uses OCR to analyse the wordle screenshot by drawing a grid and then returning the words entered and also returning how many tries it took for the user to get the answer.
